# FloatingParticles
*Makes particles that... float... duh*

### Commands

- `/floatingparticle`
  - **aliases**
    - `/fp`
  - **usage**
    - `/fp clear`
      - *deletes all floating particles in a world*
    - `/fp`
      - *add a particle of the default type (defined in config)*
    - `/fp <type name>`
      - *add a particle of the given type*
